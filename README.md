# 📊 Student Data Pipeline & Analytics Project

## 🔍 Overview
This project demonstrates an end-to-end data pipeline and analytics workflow using a simulated Student Information System (SIS) dataset. It showcases database design, API integration, data analysis, and multi-platform reporting through dashboards, Excel tools, and presentation-ready visuals.

The goal was to replicate a real-world education data environment and deliver insights in formats usable by both technical and non-technical stakeholders.

⸻

## 🧠 Key Features
- Relational database design with secure access controls  
- API-based data extraction using Python  
- Multi-language dashboard (Python + R via Quarto)  
- Excel-based KPI dashboard with pivots and slicers  
- Statistical analysis and visualizations for presentations  
- Version-controlled workflow  

⸻

## 🏗️ Project Architecture
 1. Simulated SIS Data (CSV) 
 2. Database (Supabase / PostgreSQL) 
 3. Python API Pipeline (Pandas)  
 4. Quarto Dashboard (HTML) 
 5. Excel Dashboard (KPIs)
 6. R Visualizations (Presentation)

⸻

## 🛠️ Tech Stack

- **Database:** Supabase (PostgreSQL, RLS, SQL)
- **Data Engineering:** Python (Pandas, API integration)
- **Visualization:** R (ggplot2), Plotly
- **Dashboarding:** Quarto (HTML)
- **Business Tools:** Microsoft Excel (Pivot Tables, KPIs, Slicers)
- **Version Control:** GitHub
